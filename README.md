`git clone https://github.com/64bit-integer-limit/KDEplasmaVoidLinux.git`

A personal semi-automatic bash script to install KDE plasma on void linux

* installs
    * kde-plasma
    * kde-baseapps
    * sddm
    * xorg
    * pulseaudio
    * NetworkManager

* does some service things for
    * NetworkManager
    * sddm
    * dbus

* removes
    * conflicting wpa_supplicant service
after that, it reboots your computer
